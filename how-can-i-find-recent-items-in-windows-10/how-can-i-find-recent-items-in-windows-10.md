# How can I find recent items in Windows 10?

![windows_10_logo](windows_10_logo.png)

This post answers, "How can I find recent items in Windows 10?"

## **<u><span>Steps</span></u>**

\# Type: **Windows Key + r**

\# Type: **shell:recent**

Many more "shell" examples listed @
